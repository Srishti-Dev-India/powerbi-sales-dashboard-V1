# Power BI Sales Dashboard

## Overview
Sales analytics dashboard built in Power BI showcasing KPI tracking, trend analysis, and interactive visualizations to support business decision-making.

## Key Features
- KPI tracking (Revenue, Profit, Sales Volume)
- Time-series analysis (monthly/yearly trends)
- Region and category performance breakdown
- Interactive filters and drill-down capability

## Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling

## Dataset
Sample sales dataset used for analysis (can be customized for real-world use cases).

## How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Interact with filters and visuals  

## Project Structure
- `sales-dashboard-v1.pbix` → Main report file  

## Insights
- Identified top-performing regions and categories  
- Highlighted seasonal sales trends  
- Enabled performance comparison across dimensions
